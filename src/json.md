>本接口格式声明  
>【模块功能】：  
> 【服务器地址】：【请求方法】  
> 【请求格式】{ key1 : value1, ...}  
> 【响应格式】{key1 : value1, ...}  

### 用户登录：  
http://localhost:8080/login : GET  
```
request:{
    code:res.code

}
response:{
    sessionId: id,
    userChoice: "blue|green"
}
```

### 用户注销：  
http://localhost:8080/logout : GET  
```
request:{
    sessionId: id,
    destroy: "destroy"
}
response:{
    status:"success|failure"
}
```
